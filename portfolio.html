Gross Federal Debt

My seventh chart shows Gross Federal Debt in the USA from 2012 to 2019. Instead of using an API or raw csv in my github repository, I have manually input the data points into my json file. In my json code I have used the 'monotone' function in the 'interpolate' method of the vega library. This smoothens what otherwise would have been a purely linear graph which from a visual standpoint, makes the graph look more visually appealing.

Web Scraper

This chart is a relatively basic implementation of a web scraper. Using Python, I scraped a table on tax rates sourced from Wikipedia using the pandas libary before sorting and downloading the dataframe. Due to Wikipedia's formatting with numbered referencing e.t.c, the sorting of the table was not entirely accurate and some data had to be cleaned, therefore removing an element of automation, however the concept still held strong.

'All inflation is transitory'

At the Finance in Flux talk, John Turner, stated that in some ways 'all inflation is transitory'. In this graph I have decided to assess this statement. For inflation to be transitory, it should be short-lived and should have little impact on forward inflation expectations. The chart below shows monthly consumer inflation expectations in the uSA from the University of Michigan. These are now at the highest level since around 2011, a year in which the Bank of England and Federal Reseve took a similar dovish tone to the current consensus, unwilling to change rates.

'A British Philips Curve?'

In this chart I have used asessed the relationship between Inflation and Unemployment, taking monthly data from the ONS and Stata respectively from 1992 to present. I then performed a regression in Python using the pandas, seaborn and sklearn libraries. A vega visualisation in 'JSON' format may have presented more options in terms of interactivity, however using python i was able to clean data more effectively.
